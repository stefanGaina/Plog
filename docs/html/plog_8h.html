enum e_plog_SeverityLevel_t

Enumerates the bits representing if the log severity level is enabled.

Enumerator
E_PLOG_SEVERITY_LEVEL_FATAL 

If bit is set fatal logs are enabled.

E_PLOG_SEVERITY_LEVEL_ERROR 

If bit is set error logs are enabled.

E_PLOG_SEVERITY_LEVEL_WARN 

If bit is set warn logs are enabled.

E_PLOG_SEVERITY_LEVEL_INFO 

If bit is set info logs are enabled.

E_PLOG_SEVERITY_LEVEL_DEBUG 

If bit is set debug logs are enabled.

E_PLOG_SEVERITY_LEVEL_TRACE 

If bit is set trace logs are enabled.

E_PLOG_SEVERITY_LEVEL_VERBOSE 

If bit is set verbose logs are enabled.
